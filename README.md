# Password Generator (Python)

A simple Python script that generates a random password using
digits, symbols, lowercase and uppercase letters.

Ein einfaches Python-Skript zur Erstellung eines zufälligen,
sicheren Passworts mit Zahlen, Symbolen sowie Groß- und Kleinbuchstaben.

---

## Features / Funktionen
- Generates a 12-character password  
  Erstellt ein 12-stelliges Passwort
- Uses digits, symbols, lowercase and uppercase letters  
  Verwendet Zahlen, Symbole sowie Groß- und Kleinbuchstaben
- Each character group is guaranteed  
  Jede Zeichengruppe ist garantiert enthalten
- Password is shuffled for better randomness  
  Passwort wird für bessere Zufälligkeit gemischt
